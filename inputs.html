Input Parameters

We need to propagate inputs in each of submodels. To do that we use a hierarchy of input classes that is sent recursively to the models. The input class is named accordingly to the model it corresponds to and its source is located in the same folder.

Input parameters for the electrode components

An electrode (Electrode) consists of an electrode active component (ElectrodeActiveComponent), which contains the active meterial (ActiveMaterialInputParams), and a current collector (CurrentCollector)
